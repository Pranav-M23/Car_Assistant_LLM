
# 🚗 AI Car Diagnostic Assistant LLM (Phi-2 + QLoRA)

An AI-powered car troubleshooting assistant that analyzes vehicle symptoms and OBD-II diagnostic codes to generate structured diagnostic guidance.
The system uses a fine-tuned Microsoft Phi-2 Large Language Model and provides an interactive Gradio web dashboard for users.

This project demonstrates how Large Language Models (LLMs) can be specialized for automotive diagnostics and troubleshooting assistance.

📌 Project Overview

Modern vehicles contain complex mechanical and electronic systems that may develop faults over time. However, many drivers struggle to understand:

unusual vehicle symptoms

dashboard warning lights

possible causes of engine issues

recommended next actions

This project develops an AI-based assistant that helps users interpret these problems and receive structured troubleshooting guidance.

The system combines:

Large Language Models (LLM)

Automotive diagnostic knowledge

OBD-II code interpretation

Rule-based safety checks

Interactive web interface

⚙️ Key Features
🧠 AI Symptom Diagnosis

Users enter vehicle symptoms and optional OBD codes.

The AI generates structured responses including:

Symptom Summary

OBD Interpretation

Likely Causes

Risk Level

Safe Checks

Actions to Avoid

Recommended Next Steps

🔍 OBD Code Database

Built-in searchable OBD-II diagnostic database.

Users can:

search codes

view meanings

understand diagnostic trouble codes

Example:

P0300 → Random/Multiple Cylinder Misfire
P0171 → System Too Lean
🚨 Warning Light Explanation

Dashboard warning light guide including:

Check Engine

Oil Pressure

Battery Warning

Coolant Temperature

ABS Warning

Airbag Warning

Each icon includes an explanation and safety advice.

🧰 Common Car Problems Section

Provides explanations for frequent vehicle issues such as:

engine overheating

rough idle

poor mileage

brake
